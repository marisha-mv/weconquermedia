# We Conquer Media Website

Official website for We Conquer Media LLC.

## Setup

This is a static HTML website deployed via GitHub Pages.

## Domain Configuration

To connect your custom domain:

1. Add a `CNAME` file in the root directory with your domain name
2. Configure DNS settings with your domain registrar:
   - Add a CNAME record pointing to `marisha-mv.github.io`
   - Or add A records pointing to GitHub Pages IP addresses:
     - 185.199.108.153
     - 185.199.109.153
     - 185.199.110.153
     - 185.199.111.153
3. Enable GitHub Pages in repository settings
4. Add your custom domain in GitHub Pages settings

## Deployment

The site is automatically deployed when changes are pushed to the main branch.
